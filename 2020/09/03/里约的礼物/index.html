这个旋律好上头!!!不愧是沙滩男孩

​ 阿泉和布朗(鸭子)作为领舞担当辛苦了!!

约哥: 我不知道画什么

我: 你可以画一只肥美的鸭子

约哥: 我画了

​ 这鸭子!!!腿好长!!!好骚, 放在80年代绝对是迪厅舞王.

​ 整个mv出现了很多我家的孩子, 这边突发整理一下他们可能会喜欢的音乐类型(?)

  • 布朗: 迪斯科 / 灵魂乐 / 卡朋特
  • 阿泉: 白噪音 / 爵士钢琴 / 披头士 / 奇想乐队
  • 阿任: 动画片头曲 / 古典乐 / 偶像流行乐
  • 幽灵少女: 巴萨诺瓦 / 数学摇滚
  • 透君: 巴萨诺瓦 / 轻音乐 / 视觉摇滚
  • 灰濑: 乡村音乐 / 实验性音乐 / 英伦摇滚
  • 芋头: 放克爵士 / 雷鬼乐 / 老电影配乐
  • 零栋: 蓝调爵士 / 轻音乐 / jazzhippop
  • 兔兔: 电子乐 / 古典乐 / 迷幻摇滚

布朗品味挺古老的, 是个有情怀的鸭子.

阿泉有很多盘白噪音磁带, 睡不着的时候会拿出来放一放.

幽灵少女和透君都是水手服, 住的比较靠近海所以都喜欢听巴萨诺瓦(bassanova)

幽灵少女高中看过一场数学摇滚的演出, 还挺喜欢的.

透君和灰濑是普通的高中生喜好, 透君意外的会关注形象很有趣的视觉系摇滚乐队, 灰濑偶尔会买买CD

芋头有个播放器, (原本的主人应该)是无君, 但是里面塞的都是零栋的曲子

所以两人品味比较相近, 爵士乐听得比较多, 芋头听的第一首曲子应该是昆汀电影里面的<let’s stay togather>

零栋父亲是爵士钢琴师, 他工作喜欢听轻音乐.

兔兔品味有点电波系, 喜欢效果器的音效. 品味也有点上个世纪.
